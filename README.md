# Apps and Dapps

Presentation done during the Apple Developer Academy Pedagogy Conference. Consist of a presentation, contract example (generation of a ERC-20 token) and a XCode with an iOS App interacting with the contract using the Web3swift pod (https://github.com/matterinc/web3swift)
